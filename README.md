Smart Home Automation using ESP32 and Blynk

This project implements a smart home automation system using an ESP32 microcontroller and the Blynk IoT platform. The system allows users to control home appliances remotely using a mobile application as well as manual switches.

Features

Control up to 4 home appliances

WiFi based control using Blynk mobile app

Manual switch control

Real-time device status feedback

Hardware Components

ESP32 Development Board

4 Channel Relay Module

Manual Switches

WiFi Network

Power Supply

Software Used

Arduino IDE

Blynk Library

ESP32 Board Package

Working

The ESP32 connects to WiFi and communicates with the Blynk cloud. Users can control appliances from the Blynk mobile app, while physical switches allow manual operation. The relay module controls the electrical devices.
